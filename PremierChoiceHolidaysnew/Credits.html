Security Consideration

Premier choice holiday website has got to comply with the legalisation as stated under the government act for websites. The data protection act consists of up-to-date information and must only be kept during the duration of its relevancy. The data kept must be secure and must not be transferred to another country without adequate protection. If collecting data the website should include a privacy policy stating the facts how the data is used and handled. This policy should be visible for the user to read. Information should not be shared with third-parties unless the user requests to do so as the user may feel he has been spammed.


Users should download an anti-virus to keep their personal information and computer intact from viruses and malware. The server hosting the website should be regular tested against security threats and adequate protection software should be put in place to safeguard the system. Another important matter to subject is the website being prone to SQL and PHP code injection as hackers may be able to modify the code going into the servers. A smart hacker may be able to access all the user information by typing SELECT * FROM Users WHERE UserId = 105 or 1=1. This is why it is imperative that the coding is done to a proper standard so amateur hackers are not able to hack/modify the system.
